# 🧠 AnkiMate

> **A powerful, privacy-first flashcard study tool with spaced repetition and comprehensive analytics**

[![MIT License](https://img.shields.io/badge/License-MIT-green.svg)](https://choosealicense.com/licenses/mit/)
[![Made with React](https://img.shields.io/badge/Made%20with-React-61DAFB.svg)](https://reactjs.org/)
[![Tailwind CSS](https://img.shields.io/badge/Styled%20with-Tailwind%20CSS-38B2AC.svg)](https://tailwindcss.com/)

AnkiMate is a fully client-side flashcard application that helps you master any subject using scientifically-proven spaced repetition algorithms. Built with privacy in mind, all your data stays local in your browser - no servers, no tracking, no data collection.

## ✨ Features

### 🎯 **Core Study Features**
- **Spaced Repetition Algorithm**: Smart scheduling based on your performance (1 → 3 → 7 → 14 → 30 days)
- **Deck Management**: Create, edit, and organize unlimited flashcard decks
- **Card Management**: Add, edit, and delete cards with rich text support
- **Interactive Study Mode**: Clean, distraction-free study interface with keyboard shortcuts

### 📊 **Advanced Analytics Dashboard**
- **Study Streaks**: Track consecutive days of studying with motivational indicators
- **Daily Progress**: Real-time tracking of cards studied, accuracy, and session count
- **Weekly Charts**: Visual progress bars showing your study patterns
- **Deck Performance**: Individual deck statistics and performance comparisons
- **Historical Data**: 365 days of study history and trends

### 🔒 **Privacy & Data Control**
- **100% Local Storage**: All data stays in your browser - complete privacy
- **No Server Dependencies**: Works offline, no internet required after loading
- **Export/Import**: Backup and share decks as JSON files
- **Zero Tracking**: No analytics, cookies, or data collection

### 🎨 **Modern User Experience**
- **Responsive Design**: Works perfectly on desktop, tablet, and mobile
- **Clean Interface**: Minimal, distraction-free design with Tailwind CSS
- **Accessibility**: Full keyboard navigation, ARIA labels, semantic HTML
- **Dark/Light Theme**: Comfortable studying in any lighting condition

## 📖 How to Use

### Getting Started
1. **First Time**: Open the app to find a sample math deck ready to study
2. **Create Decks**: Click "Create New Deck" to add your own study topics
3. **Add Cards**: Within a deck, add question/answer pairs
4. **Start Studying**: Click "Study Now" to begin your spaced repetition journey

### Study Workflow
1. **See Question**: Read the front of the card
2. **Think**: Try to recall the answer
3. **Reveal Answer**: Click "Show Answer" or press Enter
4. **Self-Assess**: Mark as "Correct" or "Incorrect"
5. **Repeat**: Continue with the next due card

### Analytics & Progress
- **Dashboard**: Click the 📊 Dashboard button to view detailed analytics
- **Study Streaks**: See your consecutive study days
- **Performance**: Track accuracy and improvement over time
- **Deck Stats**: Compare performance across different subjects

## 🧪 Spaced Repetition Science

AnkiMate uses a proven spaced repetition algorithm:

- **Correct Answer**: Interval increases (1 → 3 → 7 → 14 → 30 days)
- **Incorrect Answer**: Interval resets to 1 day
- **Smart Scheduling**: Only shows cards when they're due for review
- **Long-term Retention**: Optimizes for maximum memory retention

## 💾 Data Management

### Local Storage
- All decks, cards, and progress stored in browser's local storage
- Data persists between sessions automatically
- No account creation or login required

### Export/Import
- **Export**: Download any deck as a JSON file for backup
- **Import**: Upload JSON files to restore or share decks
- **Backup Strategy**: Regular exports recommended for important decks
